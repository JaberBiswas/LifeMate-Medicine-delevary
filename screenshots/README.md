# Screenshots

This folder contains screenshots of all the app screens.

## How to Add Screenshots

### Step 1: Take Screenshots
Take screenshots of each screen in your app. You can use:
- **Android Emulator**: Use the screenshot button in the emulator toolbar
- **iOS Simulator**: `Cmd + S` or Device â†’ Screenshot
- **Physical Device**: Use your device's screenshot feature
- **Flutter DevTools**: Use the screenshot feature in DevTools

1. Copy your screenshot files to this `screenshots/` folder
2. Make sure file names are descriptive and follow the naming convention above

### Step 4: Commit and Push to GitHub
```bash
git add screenshots/
git commit -m "Add app screenshots"
git push origin main
```

## Recommended Screenshot Sizes
- **Mobile**: 1080x1920 (Portrait) or 1920x1080 (Landscape)
- **Tablet**: 1536x2048 (Portrait) or 2048x1536 (Landscape)
- **Web**: 1920x1080

## Tips
- Use consistent device/emulator for all screenshots
- Ensure screenshots show actual content (not empty states)
- Crop screenshots to remove unnecessary borders
- Keep file sizes reasonable (compress if needed)
