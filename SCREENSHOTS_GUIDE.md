# How to Upload Screenshots to GitHub

## Quick Guide

### Method 1: Using Git Commands (Recommended)

1. **Take screenshots** of all your app screens
2. **Save screenshots** in the `screenshots/` folder
3. **Add to Git:**
   ```bash
   git add screenshots/
   ```
4. **Commit:**
   ```bash
   git commit -m "Add app screenshots"
   ```
5. **Push to GitHub:**
   ```bash
   git push origin main
   ```

### Method 2: Using GitHub Web Interface

1. Go to your repository: https://github.com/JaberBiswas/LifeMate-Medicine-delevary
2. Click on the `screenshots` folder
3. Click "Add file" → "Upload files"
4. Drag and drop your screenshot files
5. Scroll down and click "Commit changes"

## Screenshot Naming Convention

Use descriptive names:
- `01-welcome-screen.png`
- `02-login-screen.png`
- `admin-01-dashboard.png`

Or organize in subfolders:
- `screenshots/user/`
- `screenshots/admin/`

## After Uploading

Once screenshots are uploaded, you can:
1. Update the main README.md to display screenshots
2. Add image links in README like: `![Home Screen](screenshots/04-home-screen.png)`
