# LifeMate-Medicine-delevary

## Screenshots

Screenshots of all app screens are available in the [`screenshots/`](screenshots/) folder.

### How to Add Screenshots

1. Take screenshots of your app screens
2. Save them in the `screenshots/` folder
3. Use descriptive names like:
   - `01-welcome-screen.png`
   - `02-login-screen.png`
   - `admin-01-dashboard.png`
4. Commit and push:
   ```bash
   git add screenshots/
   git commit -m "Add app screenshots"
   git push origin main
   ```

For detailed instructions, see [SCREENSHOTS_GUIDE.md](SCREENSHOTS_GUIDE.md)
